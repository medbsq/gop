### Description:
this is  a python script allow to fetch open ports from shodan internetdb .

### Usage:

```
python3 gop.py -f <ips file> -o <output>
```

#### Options:
```
  -h, --help   show this help message and exit
  -f IPS_FILE  spicify ips_file
  -o OUTPUT    spicify output file
  -i IP        spicify ip
```
